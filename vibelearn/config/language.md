# language

關於語言
1. 內容使用繁體中文 附上必要的英文
2. 專有名詞優先保留英文原文不翻譯，若要翻譯則必須在括號中備註原文，例如: 正交(orthogonal)
3. 若原教材中使用英文縮寫教學，在撰寫時可以引用同樣的英文縮寫說明；但必需在名詞對照表中，附上其對應的全名，再對其進行解釋；例如: markdown```| ELF / COFF | COFF 全名：Common Object File Format（教材寫這個）；ELF 全名：常見標準全名是 Executable and Linkable Format；你的教材投影片上寫 Extended Linker Format（同一縮寫，不同說法）。Object file format（教材提到 COFF/ELF），描述編譯後碼與 section 的儲存結構。 |```
